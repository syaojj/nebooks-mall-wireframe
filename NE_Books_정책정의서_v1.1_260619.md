## 1. 회원 정책

### 1.2 권한 정책

| 기능 | 비회원 | 일반회원 | 교사/강사 |

## 2. 통합회원 서비스 연동 정책

### 2.1 SSO 연동

- NE Books와 NE Tutor는 통합회원 기반 SSO를 사용한다.
- 로그인 정보는 서비스 간 연동한다.

### 2.2 연동 범위

| 구분 | 연동 |
|------|:----:|
| 로그인 상태 | O |
| 회원정보 | O |
| 비밀번호 | O |
| 포인트 | O |
| 주문내역 | X |
| 학습이력 | X |
| 다운로드 이력 | X |
| 마이페이지 | X |

### 2.3 운영 원칙

- 인증은 통합회원 기준으로 운영한다.
- 서비스별 마이페이지 및 이용이력은 개별 관리한다.

### 2.4 개발 검토사항

- NE Books와 NE Tutor 간 SSO 로그인 연동 방식 확인 필요
- 로그인 유지, 로그아웃, 세션 만료 처리 방식 확인 필요
- 통합회원 회원구분(일반회원/교·강사) 수신 방식 확인 필요
- SSO 연동 시 Front 화면별 권한 처리 가능 여부 확인 필요

## 3. 교재 정책

### 3.1 품절 정책

- SAP 재고 기준으로 품절 관리
- 매일 오전 9시 SAP 재고 연동
- SAP 재고수량 < 예비수량 시 자동 품절
- 품절 상품은 구매 불가
- 품절 상품은 장바구니 담기 불가

### 3.2 관련 교재

Admin > 교재관리 > 다음단계교재 기준 노출

### 3.3 학습자료 연결

Admin > 교재관리 > 부가서비스 등록자료 기준

## 8. 배송 정책

## 10. 포인트 정책

### 10.1 포인트 체계

- NE Point 통합 운영
- B&G Point → NE Point 이전

### 10.2 적립 정책

| 항목 | 적립 |
|------|------|
| 마케팅 수신동의 | 최초 1회 2,000P |
| 교재 후기 | 구매 교재별 최초 1회 500P (글자수 등 필수 요건 충족 시) |
| 교재 구매 | 정가 기준 지급 |
| 상시 이벤트 | 3,000P ~ 20,000P |

### 10.3 사용 정책

**교재 구매**
- 사용 가능

**NE Times 구매**
- (출판교육·학습) 마케팅 수신동의 이력 보유 시
- 1회 한정 사용 가능
- 사용 범위 : 1,000P ~ 2,000P

### 10.4 유효기간

- 구매 적립 : 적립일 기준 1년
- Admin 적립 : 설정 만료일 기준

### 10.5 차감 정책

- 유효기간 임박 포인트 우선 차감
- 결제취소 시 복원
- Admin 수동 차감 가능

### 10.6 후기 포인트

- 구매 교재별 최초 1회만 적립
- 후기 삭제 시 포인트 회수
- 사용 완료 포인트는 마이너스 처리
